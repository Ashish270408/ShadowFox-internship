# ShadowFox Cybersecurity Internship

This repository contains the work completed during my one-month Virtual Cybersecurity Internship at ShadowFox (March 2026).

The internship focused on beginner-level penetration testing concepts, including reconnaissance, enumeration, and network traffic analysis using industry-standard tools.

---

## 📌 Internship Overview

- Duration: March 1 – March 31, 2026
- Domain: Cyber Security
- Environment: Kali Linux (Virtual Machine)

---

## 🛠️ Tools & Technologies Used

- Nmap (Port Scanning)
- Gobuster (Directory Enumeration)
- Wireshark (Network Traffic Analysis)
- Kali Linux
- VirtualBox
